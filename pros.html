The Pros for the Waterfall model

  • Planning and designing are more straightforward.
  • Progress is easily measured because the full scope of the work is known in advance.
  • Allows teams to work in parallel because the design of the project is completed early.
  • Prevents code from not fitting well with the rest of the project because everything is done around the same time
